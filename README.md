# PrepNexus

## Overview
PrepNexus is a career readiness platform that helps users:
- Analyze resumes
- Detect skill gaps
- Match target job roles
- Practice interviews
- Prepare for aptitude and DSA

## Current Features
- Resume PDF parsing
- NLTK-based skill extraction
- RAG-based role requirement retrieval
- Skill gap analysis
- Match score

## Tech Stack
- Python
- NLTK
- LangChain
- FAISS
- Sentence Transformers
